## Key Paths

| Purpose | Path |
|---------|------|
| Frontend entry | `index.html`, `src/main.tsx` |
| API base | `src/api/client.ts` |
| Socket hook | `src/hooks/useSocket.ts` |
| Game state | `src/store/gameStore.ts` |
| Backend entry | `backend/src/server.ts` |
| Database schema | `backend/src/db/schema.ts` |
| Shared types | `shared/types.ts` |

---

## Import Paths

- Backend imports shared: `../../../shared/types` (from `backend/src/`).
- Frontend imports shared: `../../shared/types` or via alias.
- Backend runs from repo root so `shared/` resolves correctly.
